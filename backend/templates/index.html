API Documentation

For data processing between the scraping team and front end

API Endpoint

POST /api/calculate

Expected Input Format

The API accepts data in two formats:

1. Standard Dictionary Format:
{
  "uber_rides": [
    {"distance": 17.44, "time": "56 minutes"},
    {"distance": 5.2, "time": "15 minutes"}
  ],
  "lyft": [
    {"distance": 12.3, "time": "30 minutes"},
    {"pickup_location": "123 Main St", "dropoff_location": "456 Oak St", "time": 15}
  ],
  "uber_eats": [
    // Restaurant and delivery address
    {"restaurant": "Chicken Shanty Corvallis", "delivery_address": "1845 NW Polk Ave, Corvallis, OR"},
    // Alternative format with direct distance
    {"distance": 3.7}
  ],
  "doordash": [
    // Restaurant and delivery address
    {"restaurant": "McDonalds", "delivery_address": "789 Home St, City"},
    // Alternative format with direct distance
    {"distance": 2.5}
  ],
  "flights": [
    // Using segments format (preferred)
    {"segments": [
      {"origin": "PDX", "destination": "LAX"}
    ]},
    // Multi-segment flight example
    {"segments": [
      {"origin": "PDX", "destination": "SEA"},
      {"origin": "SEA", "destination": "JFK"}
    ]},
    // Alternative format with direct distance
    {"distance": 2500}
  ]
}
2. Quickstart Format (List Format):
[
  {"type": "Uber Ride", "distance": "17.44", "time": "56"},
  {"type": "Lyft Ride", "time": 15, "pickup_location": "123 Main St", "dropoff_location": "456 Oak St"},
  {"type": "Uber Eats", "restaurant": "Chicken Shanty Corvallis", "delivery_address": "1845 NW Polk Ave, Corvallis, OR"},
  {"type": "Door Dash Order", "restaurant": "McDonalds", "delivery_address": "Empire State Building, New York, NY"},
  {"type": "flight", "segments": [
    {"origin": "PDX", "destination": "LAX"},
    {"origin": "LAX", "destination": "JFK"}
  ]}
]
Google API Integration

For address-based and airport-based distance calculations, make sure to:

  1. Install the Google Maps library: pip install googlemaps
  2. Set your API key as an environment variable: export GOOGLE_MAPS='your-key-here'

Without this setup, the API will still work with direct distance values, but not with address or airport calculations.
